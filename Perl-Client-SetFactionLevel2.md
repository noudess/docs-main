**Name**|**Type**|**Description**

faction_id||
char_class||
char_race||
char_deity||

### Example

```perl
my $char_id = 1;
my $faction_id = 1;
my $char_class = 1;
my $char_race = 1;
my $char_deity = 1;
my $value = 1;
my $temp = 1;

$client->SetFactionLevel2($char_id, $faction_id, $char_class, $char_race, $char_deity, $value, $temp); # Returns void
```
