sets a corpse cash.
### Arguments
**Name**|**Type**|**Description**
:---|:---|:---
in_copper||
in_silver||
in_gold||
in_platinum||

### Example

```perl
my $in_copper = 1;
my $in_silver = 1;
my $in_gold = 1;
my $in_platinum = 1;

$corpse->SetCash($in_copper, $in_silver, $in_gold, $in_platinum); # Returns void
```
